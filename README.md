Battleship-AI
=============

IRC based Battleship game to encourage programming basic AI clients.

Authors: CaptSullivan, debus

Contributors:


Get Started:
------------

Download interface.py and a sample AI. Compile the AI and execute interface.py as:

```
./interface.py ./ai
```
